# Simple Circular Buffer Delay

A lightweight Python script that generates a synthesized audio signal and applies a custom-built circular buffer delay effect. 

This project demonstrates how real-time audio processing works under the hood by processing audio sample-by-sample, mimicking the behavior of C++ DSP (Digital Signal Processing) audio plugins.

## ⚙️ Prerequisites
To run this script, you will need Python installed along with the `numpy` and `scipy` libraries. You can install the required dependencies using pip:

```bash
pip install numpy scipy
